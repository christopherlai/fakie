# Fakie

Ruby [libphonenumber](http://code.google.com/p/libphonenumber) wrapper with [ExecJS](https://github.com/sstephenson/execjs).

## Installation

Add this line to your application's Gemfile:

``` ruby
gem 'fakie'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fakie
